# Qualitative Codebook

Based on the interview data, we presented a qualitative codebook to reflect the taxonomy of threat vectors, verification mechanisms, and systemic constraints, as shown in Table 1.

### Table: Qualitative codebook for GenAI-enabled refund fraud analysis.

| Theme | Code | Description |
| :--- | :--- | :--- |
| **Threat Vectors (RQ1)** | **Automated promotion and subsidy arbitrage** | Exploiting promotional algorithms and shipping subsidies using coordinated bulk accounts. |
| | **Synthetic persona construction** | Utilizing GenAI to fabricate human-like behavioral histories and profiles to evade detection. |
| | **Entry barrier reduction** | Lowering the technical expertise required for systematic fraud through accessible AI generation tools. |
| | **Hyper-realistic defect fabrication** | Synthesizing physical damage that respects material properties and physical logic to deceive visual inspection. |
| | **Contextual attribute forgery and digital recycling** | Forging communication records and reusing identical synthetic evidence across multiple stores simultaneously. |
| | **Algorithmic bypass and asynchronous substitution** | Exploiting automated chatbots to secure immediate refunds before merchants can physically inspect returns. |
| | **Carrier-side interception** | Rerouting shipments mid-transit after prematurely securing an automated platform refund. |
| | **Off-platform double extortion and entrapment** | Weaponizing fabricated content publicly or migrating to external channels to extract redundant refunds. |
| | **Regulatory exploitation and psychological coercion** | Leveraging administrative complaints and emotional pressure to force merchants into financial settlements. |
| **Verification & Resolution (RQ2)** | **Manual inspection and heuristic judgment** | Evaluating evidence authenticity through physics consistency, spatial interrogation, and collaborative peer consensus. |
| | **Technical and automated authentication** | Deploying algorithmic risk screening, metadata forensics, and cross-referencing provenance markers. |
| | **Contextual and evidentiary verification** | Cross-referencing logical narratives, temporal data (e.g., live video), and historical account reputations. |
| | **Platform-level criteria** | Adjudicating disputes via automated triage, weighted decision metrics, and asymmetric procedural rules. |
| | **Merchant-level criteria** | Filtering and addressing claims based on operational scale, historical transaction data, and physical plausibility. |
| | **Negotiation and direct settlement** | Requesting physical returns, multi-angle proof, or alternative remedies to distinguish authentic dissatisfaction from fraud. |
| | **Escalation and external recourse** | Triggering supervisory reviews, specialized data reporting, or law enforcement intervention for complex disputes. |
| | **Adjustment and loss absorption** | Internalizing low-value dispute costs or executing partial compensation to minimize operational and emotional friction. |
| **Systemic Constraints (RQ3)** | **Advanced AI outpaces detection** | The visual fidelity of synthetic evidence exceeding the capabilities of current manual heuristics and detection tools. |
| | **Missing ground truth** | The structural inability to establish pre-shipment documentation, particularly limiting defenses for perishable goods. |
| | **Privacy and data fragmentation** | Legal and compliance boundaries preventing the establishment of cross-platform intelligence sharing and centralized registries. |
| | **Structural bias and irreversibility** | Platform adjudication frameworks that structurally favor consumers and lack mechanisms for retroactive correction. |
| | **Bottlenecks and automation bias** | High dispute volumes forcing reliance on automated systems that frequently misclassify forged artifacts as genuine quality issues. |
| | **Suppressed merchant discourse** | Algorithmic moderation restricting merchants' ability to actively contest claims or compel evidence collection. |
| | **Economic imbalance in proof production** | The disproportionate labor and procedural cost required for merchants to disprove near-zero cost synthetic fraud. |
| | **Reputational risk and incentive misalignment** | The continuous threat of negative ratings coercing merchants into accepting fraudulent claims as a standard operational cost. |
